# composer 教程
https://docs.phpcomposer.com/

## 使用
```
1. curl -sS https://getcomposer.org/installer | php
    下载 composer.phar 或复制其它项目的
2. php composer.phar install
    处理 composer.json 依赖关系，并把其安装到 vendor 目录下
```
